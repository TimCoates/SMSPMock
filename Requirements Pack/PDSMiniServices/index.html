Overview: PDS Mini Services Domain Message Specification

The PDS Mini Services Domain Message Specification supports the following services:

  • verifyNHSNumber
  • getNHSNumber
  • getPatientDetailsByNHSNumber
  • getPatientDetailsBySearch
  • getPatientDetails
